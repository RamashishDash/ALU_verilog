# ALU_verilog
Using verilog designing a arithmetic logic unit.
## An ALU (Arithmetic Logic Unit) performs arithmetic and logical operations like:

    1. Addition
    2. Subtraction
    3. AND
    4. OR
    5. XOR
    6. NOT
    7. Comparison
```
| Select | Operation   |
| ------ | ----------- |
| 000    | A + B       |
| 001    | A - B       |
| 010    | A AND B     |
| 011    | A OR B      |
| 100    | A XOR B     |
| 101    | NOT A       |
| 110    | Left Shift  |
| 111    | Right Shift |
```
